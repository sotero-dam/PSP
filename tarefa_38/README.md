# TAREFA 38 - CONTRASINAL

**Desenvolvido por**: Sofía Otero

**Asignatura**: Programación de Servizos e Procesos (PSP)

**Versión**: 1.0

Este programs xera hashes SHA-256 para verificar a integridade de 
calquera texto ou ficheiro. Ao ser un proceso irreversible,
permite validar credenciais e detectar modificacións sen
necesidade de gardar nin expoñer os datos orixinais.
